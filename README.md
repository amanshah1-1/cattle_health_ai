# 🐄 Cattle Health AI Assistant

A multi-agent AI assistant for cattle health using **LangChain RAG** + **Ollama** + **FAISS embeddings**.  

Supports:
- Nutrition advice  
- Disease diagnosis  
- Treatment recommendations  

Also provides:
- Streamlit interface for interactive chat  
- FastAPI endpoint for integration  
- Logging of all queries  

## 🚀 How to Run

### Streamlit
```bash
pip install -r requirements.txt
streamlit run app.py
